GALLERY
  • git config --global user name "your name" (один раз в жизни)
  • git config --global user email "your email" (один раз в жизни)
  • git init - создать репозиторий (один раз для проекта)
  • git add index.html - добавил файл index.html
  • git commit -m "Создал файл index.html"
  • git log (необязательно) - для просмотра истории коммитов
  • git checkout ... (необязательно) - переключиться на коммит
  • git status (необязательно) посмотреть статус репозитория
  • git remote add origin https://... - устанавливается подключение к удаленному серверу и git репозиторию на нем.
  • git push origin master - пушим изменения на ветку в удаленный репозиторий
